# Heart-disease-prediction-project-using-ml

---
**Title:** `Cardiovascular-Disease-Prediction`

**Description:**
A machine learning project designed to predict the likelihood of cardiovascular disease based on clinical health indicators. The project utilizes various classification algorithms to analyze risk factors and evaluate model performance.

**Key Features:**
* **Data Analysis:** EDA on clinical datasets.
* **Models:** Implementation of Random Forest, Gradient Boosting, SVM, and K-Medoids.
* **Performance:** Comprehensive evaluation using accuracy, precision, and recall metrics.
